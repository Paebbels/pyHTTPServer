pyHTTPServer Documentation
##########################

An HTTP Server implementation offering a pyHTTPInterface.


Installation/Updates
********************

Installation using PIP
======================

.. code-block:: bash

   pip3 install pyHTTPServer


Updating using PIP
==================

.. code-block:: bash

   pip3 install -U pyHTTPServer
